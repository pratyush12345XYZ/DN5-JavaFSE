Module Objective
================

The objective of this module is to gain practical knowledge of advanced testing techniques using JUnit 5. The exercises demonstrate how to write efficient, maintainable, and professional unit tests using Maven and JUnit Jupiter. The module covers parameterized tests, test suites, execution ordering, exception handling, and performance testing.
